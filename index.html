Cara Membuat Template Word Custom
  1. Buat dokumen SKL di Microsoft Word dan simpan sebagai format .docx
  2. Tulis placeholder seperti {{NAMA}}, {{NISN}} di posisi yang diinginkan
  3. Upload file .docx tersebut di form di atas
  4. Sistem akan otomatis mengisi data siswa dan siswa dapat men-download SKL sebagai file Word (.docx)
Sistem membaca template Word Anda, mengganti semua placeholder dengan data siswa, lalu menghasilkan file .docx siap unduh untuk setiap siswa yang lulus.
